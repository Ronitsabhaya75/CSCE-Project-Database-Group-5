# Dealer Interface (Vehicle Locator)
**Owner:** Snehitha Paruchi

Your task is to build the CLI menu for dealership employees. 
You need to write SQL queries in your code that execute the following:
1. **View Local Inventory:** Query `Inventory` filtering by a specific `dealer_id`.
2. **Global Vehicle Locator:** Query `Inventory` across all dealers for a specific VIN/Model.
3. **Process Sale:** Execute an `INSERT INTO Sale` and an `UPDATE Inventory SET status = 'sold'`.
