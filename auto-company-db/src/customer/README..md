# Online Customer Interface
**Owner:** [Name 2]

Your task is to build the CLI menu that allows a customer to browse the database. 
You need to write SQL queries in your code that execute the following menu options:
1. **Search Dealerships by Location:** Query the `Dealer` table by city/state.
2. **Browse Vehicle Catalog:** Query the `Brand`, `Model`, and `Options` tables.
3. **Search Inventory:** Query the `Inventory` table joined with `Dealer` to find a specific car model and its price.
